# 终端节点<a name="sis_03_0004"></a>

终端节点即调用API的请求地址，不同服务不同区域的终端节点不同。

目前语音交互服务一句话识别、录音文件识别支持以下地区和终端节点：

>![](public_sys-resources/icon-note.gif) **说明：** 
>当前服务仅支持北京和上海区域，后续会陆续上线其他区域。
>-   华北-北京一，该区域资源有限，当前仅支持已选择该区域的老用户使用，新用户不可见，后续该区域不可用。
>-   华北-北京四，推荐的区域，支持一句话识别、录音文件识别、实时语音识别、语音合成和口语测评和热词等接口。
>-   华东-上海一，推荐的区域，支持一句话识别、录音文件识别、实时语音识别、语音合成和热词等接口 。
